Grinnell theme
==============

D6 Fusion sub-theme for Grinnell.


Instructions:
-------------

- download and enable fusion 6.x-1.13 or later
- navigate to the themes folder and do
- git clone https://github.com/DGI-GRINNELL/grinnell-theme.git
- mv grinnell-theme grinnell
- enable the theme
- the main navigation menu is populated with the primary links
- the top right corner of the header will be populated with the secondary links

In the Grinnell theme settings set where X = checked and O = unchecked
X Logo
X Site name
O Site slogan
O Mission statement
O User pictures in posts
O User pictures in comments
O Search box
X Shortcut icon
X Primary links
X Secondary links
