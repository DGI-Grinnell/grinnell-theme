Grinnell theme
==============

D6 Fusion sub-theme for Grinnell.


Instructions:
-------------

- download and enable fusion 6.x-1.13 or later
- navigate to the themes folder and do
- git clone https://github.com/DGI-GRINNELL/grinnell-theme.git
- mv grinnell-theme grinnell
- enable the theme
- the main navigation menu is populated with the primary links
- the top right corner of the header will be populated with the secondary links

In the Grinnell theme settings set where X = checked and O = unchecked
X Logo
X Site name
O Site slogan
O Mission statement
O User pictures in posts
O User pictures in comments
O Search box
X Shortcut icon
X Primary links
X Secondary links 

Fusion settings:
- you can choose a fixed or fluid layout and set the width of the sidebars
under: Fusion theme settings > General settings > Layout

Custom Grinnell theme settings:
- Enable contact form in footer (enable contact module)
- Enable Islandora simple search (enable islandora_solr_search)
- Upload custom banner (replaces logo and site name)
- Ability to stretch banner to full width
- Override color of links and the islandora solr search button
